# Keyboard input

Los Keyboard Input se definen entre doble corchetes `[[` `]]`.

```demoCode[markdown]

[[a]] [[3]]

```

Algunas teclas se sustituyen por su símbolo automáticamente.

```demoCode[markdown]

[[alt]] [[command]] [[shift]] [[ctrl]]

```

También se pueden agrupar varias teclas

```demoCode[markdown]

[[command C]] Copiar
[[command V]] Pegar


```
