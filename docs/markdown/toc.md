# Tabla de contenidos

Añade una tabla de contenidos. Se genera de forma automática con las cabeceras de segundo nivel `h2` `## Título`.

```demoCode[markdown]
@[toc]

## Título 1

Lorem ipsum ...

## Título 2

Lorem ipsum ...

## Título 3

Lorem ipsum ...

## Título 4

Lorem ipsum ...
```
